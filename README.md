## 关于

AmazeCard是一个在线工具，可以将文字和表情符号转换为图片。它可以自定义生成图像的颜色和卡片样式，以及自定义水印（文本或logo），还可以在生成的图像中放置链接并形成相应的二维码。

## 功能

基于文本和表情符号生成精美图片

## Advantage

1. 🎈自定义专属水印/logo
2. 🎃丰富的表情符号图案
3. 👓实时渲染效果
4. 🎨定制各种卡片样式
5. 🕹可以根据提供的链接在图像中生成相应的二维码
